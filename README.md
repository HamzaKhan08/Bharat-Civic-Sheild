# 🇮🇳 Bharat Civic Shield
### Advanced AI-Powered Fake News Detection & Civic Intelligence Platform

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![AI](https://img.shields.io/badge/AI-NLP%20%7C%20Machine%20Learning-green)
![Stack](https://img.shields.io/badge/Stack-MERN%20%2B%20Python-orange)
![Status](https://img.shields.io/badge/status-Active%20Development-brightgreen)

**Bharat Civic Shield** is an advanced AI-powered platform designed to detect misinformation, fake news, propaganda, and misleading narratives circulating on the internet.

The platform leverages **Natural Language Processing (NLP), machine learning, and intelligent fact-checking systems** to analyze articles, headlines, claims, and media content, helping users verify credibility and understand misinformation patterns.

The goal is to strengthen **digital literacy, civic awareness, and responsible information consumption**.

---

# 🌍 Vision

In the digital age, misinformation spreads faster than truth.  
**Bharat Civic Shield** aims to provide citizens with tools to verify information and understand the reliability of content they encounter online.

The platform promotes:

- Truthful information
- Responsible journalism
- Civic awareness
- Critical thinking

---

# 🚀 Key Features

## 🔎 AI Fake News Detection

The platform uses advanced **Natural Language Processing models** to classify news content as:

- True
- False
- Misleading
- Manipulated
- Unverified

The system evaluates:

- linguistic patterns
- emotional manipulation
- propaganda indicators
- logical inconsistencies
- misinformation signals

---

## 📄 Article & Claim Analyzer

Users can submit:

- News articles
- Website URLs
- Headlines
- Social media posts
- Claims

The AI analyzes the content and generates a **credibility score and explanation**.

---

## 🧠 AI Explanation System

Unlike simple classifiers, the platform explains *why* the content may be misleading by highlighting:

- suspicious phrases
- emotional manipulation
- unreliable sources
- factual inconsistencies

---

## 🖼️ Image & Media Verification

The system can analyze image URLs and detect:

- reused images
- misleading context
- manipulated visuals

---

## 📊 Credibility Scoring System

Each analyzed piece of content receives a **credibility score based on multiple factors**:

- source reputation
- language patterns
- factual alignment
- publication credibility

---

## 👥 Community Reporting

Users can report suspicious content for review.  
The system combines **AI analysis + community feedback**.

---

## 📚 Civic Education

The platform also provides educational resources explaining:

- misinformation tactics
- propaganda strategies
- media bias
- digital literacy

# 📊 AI Analysis Pipeline

1. User submits claim or article  
2. Text preprocessing  
3. NLP feature extraction  
4. Model classification  
5. Source credibility analysis  
6. Fact comparison with knowledge sources  
7. Generate credibility score  
8. Present explanation to user

# 🚀 Getting Started

## Clone the Repository

```bash
git clone https://github.com/yourusername/bharat-civic-shield.git
```

---

## Install Dependencies

### Frontend

```bash
cd frontend
npm install
```

### Backend

```bash
cd backend
npm install
```

### AI Service

```bash
cd ai-service
pip install -r requirements.txt
```

---

## Run the Application

Start backend server:

```bash
npm run server
```

Start frontend:

```bash
npm run dev
```

Start AI service:

```bash
python app.py
```
